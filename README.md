# Invoice-System
> Invoice System Project for Mr. Erik Farrar built with Node.js, Express, Handlebars, and MongoDB

## Installation

Any OS:

After installing Node.js:

Import files...

```sh
npm install body-parser express express-handlebars mongoose nodemon
```
Then run:

```sh
npm run start-watch
```
